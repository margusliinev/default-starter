# Default Starter

Default Starter is a template for new projects built with my preferred stack.

## Tech Stack

- **TypeScript**
- **NestJS**
- **React**
- **Vite**

## Local Development

```bash
# Start database
docker compose up -d

# Install dependencies
npm install

# Run database migrations
npm run migration:run

# Start development server
npm run dev
```

## Formatting & Testing

```bash
# Formatting
npm run format

# Testing
npm run test
```
